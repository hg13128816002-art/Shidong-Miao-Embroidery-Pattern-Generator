# 施洞苗绣纹样生成器

一个零依赖的交互网页原型，用来根据纹样元素、色板、构图和刺绣质感组合生成施洞苗绣风格的数字纹样。

## 本地运行

```bash
npm run dev
```

打开 `http://localhost:5173`。

## 当前功能

- 8 类纹样元素：龙纹、鱼纹、蝴蝶纹、鸟纹、花纹、水纹、牛角纹、人物纹
- 4 种构图：团花、四方连续、围腰、清水江
- 5 套色板：节庆红、清水江、母花素稿、银饰映红、姊妹节
- 4 种质感：双针锁绣感、破线绣感、叠绣感、织锦颗粒
- 随机生成、参数化分享链接、PNG 下载
- 根据当前纹样组合展示文化说明

## 上线方式

这是静态网页，可以直接部署到 Vercel、Netlify、GitHub Pages 或任意静态托管服务。

Vercel 推荐设置：

- Framework Preset: `Other`
- Build Command: 留空
- Output Directory: `.`
